## 1. 배화교 전체 위계 체계

### 1-1. 사제 위계 (조로아스터교 실제 직책 기반)

배화교 정규 사제 위계는 조로아스터교 실제 사제 계급을 한국어로 변환한 구조.

| 배화교 단계 | 한국어 위계 | 조로아스터교 원어 | 의미 |
|---|---|---|---|
| 외문 | 보사(補師) | Hērbed | 최하위 보조 사제 |
| 내문 | 주사(主師) | Mobed | 정식 사제 (의식 주관 가능) |
| 사원 | 호법(護法) | Dastūr | 고위 사제, 사원 관장 |
| 심처 보스 | 배화교주 | Mobedān Mobed | 사제들의 사제 (최고위) |
| 심처 보스 Phase 2 | 불씨를 돌보는 자 | Ātar-vaxš | 불을 살리는 자, 사제의 원형 |

**위계 접미사 규칙**:
- 검/화/경/심 등 분야 한 글자 + 위계 접미사 (보사/주사/호법)
- 사원 일반은 4글자로 확장 ("염검호법", "송경호법" 등)

### 1-2. 무력 조직 위계 (사제와 별개)

배화교 내부에 사제 계열과 **병렬로 존재**하는 무력·치안 조직.

**호교당(護敎堂)** — 교단 수호, 정면 무력
- 외문: 호위 (단순 경비)
- 내문: 호교무사
- 사원: 호교통령

**계율원(戒律院)** — 이단 색출·처단, 교 전체 감시
- 내문: 감찰사자
- 사원: 계율원주

두 조직 모두 외법(권법) / 보법 계통 무공 드랍.

### 1-3. 보스 위계

각 단계 보스는 그 단계의 최고 권위자.

| 단계 | 보스 호칭 | 원어 병기 |
|---|---|---|
| 외문 | 배화교 외문수좌 | (Hērbed) |
| 내문 | 배화교 내문상좌 | (Mobed) |
| 사원 | 배화교 호법신장 | (Dastūr) |
| 심처 | 배화교주 → 불씨를 돌보는 자 | (Mobedān Mobed) → (Ātar-vaxš) |

**보스 원어 표기 규칙**: 괄호 안에 소문자 원어 병기. 위계를 한국어와 원어 양쪽으로 이중 표기.

### 1-4. 히든 위계 — "아후라 마즈다의 ~" 시리즈

교주 직속 비밀 조직. 배화교 정규 위계와 별개로 존재. 외문~심처 각 단계에 히든으로 배치.

| 단계 | 이름 | 루비 원어 |
|---|---|---|
| 외문 히든 | 아후라 마즈다의 그림자 | Ahura Mazda |
| 내문 히든 | 아후라 마즈다의 눈 | Ahura Mazda |
| 사원 히든 | 아후라 마즈다의 의지 | Ahura Mazda |
| 심처 히든 | 아후라 마즈다의 화신 | Ahura Mazda |

**의미 구조**: 그림자(신체에서 파생된 것) → 눈(중요한 신체) → 의지(신체 자체) → 화신(신 자체)
- 추상화의 점층적 진화
- 모든 히든이 "Ahura Mazda" 루비를 공유 → 시리즈임이 시각적으로 즉각 읽힘

### 1-5. 지하 네임드

심처 이후의 최종 구간. 배화교 본체마저 초월하는 존재들.

| 이름 | 루비 원어 |
|---|---|
| 최초의 불꽃 자라투스트라 | Zaraθuštra |
| 아지다하카 | Aži Dahāka |

## 3. 무공 가지별 드랍·등장 구조

배화교 비급 4가지(검법/심법/보법/외법)가 각 단계에서 어느 몬스터에게서 드랍되는지.

| 가지 | 외문 | 내문 | 사원 | 심처 |
|---|---|---|---|---|
| 검법 | 검보사 | 검주사 | 염검호법 | 동방천왕 |
| 심법 | 경보사 | 경주사 | 송경호법 | 북방천왕 |
| 외법(화염) | 화보사 | 화주사 | 염검호법(융합) | 동방천왕 |
| 외법(권법) | (호위?) | 호교무사 | 호교통령 | 남방천왕 |
| 보법 | (미정) | 감찰사자 | 계율원주 | 서방천왕 |

**미정 사항**:
- 외문에 권법/보법 계열 드랍원 있음/없음 (현재는 5종 유지로 권법은 호위가 약하게 드랍, 보법은 내문부터)
- 사원 염검호법이 검법+화염 둘 다 드랍하는 구조 OK

## 4. 서사적 설정 (이번 세션 확정)

### 4-1. 아후라 마즈다의 그림자 시리즈

- **조직 정체**: 배화교 교주 직속 비밀 수호단. 외부엔 존재 비공개
- **구성**: 모든 구성원이 같은 조직명("아후라 마즈다의 ~") 하에 위계 분화
- **역할**: 배화교 정규 위계(사제/호교당/계율원)와 완전 별개로 활동
- **플레이어 체험**: 각 단계 히든으로 반복 조우하며 조직의 크기를 체감

### 4-2. 심처 답파 구조 서사

4대천왕(동→남→서→북) 순차 처치 후 중앙의 배화교주와 대면.
- 동방(일출·빛의 시작) → 남방(정오·양강) → 서방(석양·그림자) → 북방(밤·침묵) → 중앙(성화의 자리)
- 배화교/조로아스터교의 하루 순환·방위 우주론과 조응
- 답파 자체가 성화 순례 의식의 형태

### 4-3. 배화교주 페이즈 전환 서사

- **Phase 1 (배화교주, Mobedān Mobed)**: 후대 제도에서 구축된 조직적 권력의 정점. 표면적 정체
- **HP 임계치 돌파 시 전환**: 성화가 몸에서 피어오르며 본래 모습 드러남
- **Phase 2 (불씨를 돌보는 자, Ātar-vaxš)**: 조로아스터교 최초의 사제 원형. 태곳적부터 불만 지켜온 존재
- **의미**: 후대 껍데기(교주)가 벗겨지고 원초적 본질(불지기)이 드러남
- **지하 네임드와 연결**: Ātar-vaxš의 근원이 "최초의 불꽃 자라투스트라"

### 4-4. 지하 네임드

- **최초의 불꽃 자라투스트라**: 배화교 교주의 무공을 강화한 형태 + 고유 패턴. 교주의 무공이 사실은 이 자의 계승임이 드러남
- **아지다하카**: 완전 고유 패턴. 조로아스터 이원론의 악(앙그라 마이뉴) 계열
- **난이도**: 배화교 답파 후에도 즉시 잡을 수 없는 수준. 회귀 시스템 or 다음 전장 진척 후 공략

## 5. UI 표기 규칙

### 5-1. 원어 표기 방식

**보스 — 괄호 병기**:

**히든·네임드·페이즈 전환 — 루비 표기 (HTML <ruby>)**:

```
       Mobedān Mobed
배화교주

       Ātar-vaxš
불씨를 돌보는 자

       Zaraθuštra
```

### 5-2. 루비 구현 방식 (v0.4 UI 구현 프롬프트에 추가 필요)

- HTML `<ruby><rt>` 태그 사용 (크로스 브라우저 네이티브 지원)
- 몬스터 데이터에 `rubyMap` 필드 추가:
```typescript
{
  id: 'ahura_mazda_shadow',
  name: '아후라 마즈다의 그림자',
  rubyMap: { '아후라 마즈다': 'Ahura Mazda' }
}
```
- 몬스터 이름 렌더링 컴포넌트에서 rubyMap 처리
- 전투 로그·체력바 등 이름이 등장하는 모든 UI에 적용 필요
- 루비 위쪽 공간을 고려해 line-height/padding 조정

## 6. 참고한 실제 자료

### 6-1. 조로아스터교 사제 위계
- **Hērbed (Ervad)**: 최하위 사제, 주요 의식에선 보조만 가능
- **Mobed**: 정식 사제, Yasna 등 고등 의식 주관 가능
- **Dastūr**: 고위 사제, 사원 관장 (주교격)
- **Mobedān Mobed**: "사제들의 사제", 최고위. 역사적으로 Kartir(3세기)가 받은 칭호
- **Ātar-vaxš**: "불을 살리는 자", 성화 의례 담당자
- **Aθravan**: 아베스탄어 "불지기", 고대 조로아스터교 사제의 원형

### 6-2. 조로아스터교 신격
- **아후라 마즈다(Ahura Mazda)**: 최고신, 선의 근원. 히든 시리즈 핵심
- **자라투스트라(Zaraθuštra)**: 예언자·창시자. 지하 네임드
- **아지다하카(Aži Dahāka)**: 악룡, 앙그라 마이뉴의 창조물. 지하 네임드

### 6-3. 무협 마교 관용 (김용 명교 참조)
- 교주 - 좌우광명사자 - 사대호교법왕 - 오산인 구조
- 성화령: 페르시아 명교의 신물
- 호교무사, 집행사자, 전령사자 등 직책 관용어
- 배화교=마니교=명교 계보 (페르시아 기원)

### 6-4. 불교 사천왕 구조
- 동방 지국천왕, 남방 증장천왕, 서방 광목천왕, 북방 다문천왕
- 제석천 아래 방위 호법신
- 심처 사대천왕 구조에 방위 개념만 차용 (이름은 독자 작명)

## 7. 다음 세션 작업 목록 (수치 설계)

이름·컨셉이 확정됐으니 다음 세션은 전적으로 **수치**에 집중.

### 7-1. 우선순위 1: 외문 몬스터 스펙

기존 난이도 곡선:
- 흑풍채 일반 (8~9등급): HP 1550~2100
- 흑풍채 보스 (10등급): HP 4500
- 흑풍채 히든 (11등급): HP 7500

배화교 외문은 흑풍채 이후 독립 대형 던전. 세 가지 접근:
- **Option A**: 12등급부터 (HP 5700+) — 서사 우선
- **Option B**: 10등급부터 (HP 2800~3000) — 접근성 우선
- **Option C**: 11등급부터 (HP 4000~4200) — 절충

각 몬스터별 역할과 등급 배정 (안):
| 몬스터 | 역할 | 예상 등급 |
|---|---|---|
| 행자 | 최약체 | 외문 일반 최하 |
| 호위 | 단순 전투원 | 중간 |
| 검/화/경보사 | 사제 (무공 드랍) | 중간~중상 |
| 외문수좌 | 보스 | 보스급 |
| 아후라 마즈다의 그림자 | 히든 (내문급 실력) | 히든급 |

### 7-2. 우선순위 2: 검법 가지 노드 효과 수치

v0.4 메모에서 검법 가지 특수 구조 확정됨:
```
1단계 (4개): 초식 배율 / 절초 / 치명타 확률 / 치명타 피해
2단계 (2개): 절초 강화 1 / 공격 속도
3단계 (2개): 절초 강화 3 / 절초 강화 4
```

결정해야 할 수치:
- 각 노드 레벨당 효과량 (1단계 50레벨 기준 총 강화량)
- 1단계 / 2단계 / 3단계 노드 상대적 비중
- 빌드 목표 DPS (삼재검법 풀스펙 대비 몇 배?)

### 7-3. 우선순위 3: 드랍 테이블·정수 비율

- 외문 몬스터별 정수 드랍률
- 비급 드랍률 (정수보다 낮게)
- 정수 1포인트 = 노드 몇 레벨 비율
- 비급 1개 = 정수 N개 등가

### 7-4. 우선순위 4: 보스 패턴 설계

각 보스의 스킬셋:
- 외문수좌 (Hērbed): 2~3개 스킬, 야산 보스급 복잡도
- 내문상좌 (Mobed): 4개 스킬, 객잔 보스급
- 호법신장 (Dastūr): 6개 스킬, 흑풍채 보스급
- 배화교주 → 불씨를 돌보는 자: 페이즈 전환 (2페이즈, 각 5~6개 스킬, 흑풍채 히든급 이상)

히든 패턴:
- 아후라 마즈다의 그림자/눈/의지: 시리즈 연속성 있는 패턴 (단계별 강화)
- 아후라 마즈다의 화신: 신적 체급, 영역·디버프 중심
