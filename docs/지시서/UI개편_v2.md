## 1. 디자인 원칙

### 1.1 전체 톤
- 배경: #141420 (어둡되 답답하지 않게)
- 카드/패널: rgba(255,255,255,0.04) 배경 + 1px solid rgba(255,255,255,0.07) 테두리. border-radius: 14px.
- 모든 UI는 반투명. 배경을 완전히 가리지 않는다.
- 여백 넉넉하게. padding 16px, 카드 간 gap 12px.
- 모바일 게임처럼 세련되고 현대적. 대시보드/관리 화면 느낌 금지.

### 1.2 색상 (극도로 절제)

| 용도 | 색상 | 적용 대상 |
|------|------|----------|
| 기본 텍스트 | rgba(255,255,255,0.9) | 일반 텍스트 |
| 보조 텍스트 | rgba(255,255,255,0.35) | 라벨, 부가 정보 |
| 흐린 텍스트 | rgba(255,255,255,0.2) | 비용, 힌트, ??? |
| 금색 | rgba(255,215,0,0.9) | 경지, 성급 별, 활성 탭, 특수 로그 |
| 파란색 | rgba(100,180,255,0.85) | 내공 수치, 심법 |
| 초록색 | rgba(90,255,150,0.8) | HP, 회복 |
| 빨간색 | rgba(255,100,100,0.7) | 적 HP, 포기 버튼 |
| 회색 | rgba(255,255,255,0.3) | 시스템 메시지, 비활성 |

이 외 색상 금지. 형광, 원색, 네온 절대 금지.

### 1.3 폰트
- Noto Sans KR만 사용. weight: 300/400/500/600.
- Noto Serif KR 사용하지 않음 (산세리프 통일).
- 숫자: font-variant-numeric: tabular-nums.
- 본문 13px, 라벨 11px, 큰 수치 32px, 중간 수치 14px.
- index.html에 Google Fonts CDN 추가.

### 1.4 버튼
- 기본: transparent 배경, 1px solid rgba(255,255,255,0.08), border-radius 8px.
- 호버: rgba(255,255,255,0.06), border-color rgba(255,255,255,0.15).
- 비활성: opacity 0.3.
- 눈에 띄는 색상 버튼 금지.

### 1.5 탭바
- fixed bottom. background rgba(20,20,32,0.97). backdrop-filter blur(12px).
- border-top: 1px solid rgba(255,255,255,0.04).
- 4탭: 내공/경맥 | 무공/능력 | 전장 | 업적
- 아이콘(15px) + 텍스트(9px). 비활성 rgba(255,255,255,0.2), 활성 금색.
- 활성 탭 상단 2px 금색 라인.
- 아이콘: ☯ ⚔ ⛰ ★

### 1.6 스탯 컬러 인디케이터
- 선기: #60a5fa (6x6 dot) — 경맥 투자에서만
- 경신: #a78bfa (6x6 dot)
- 마기: #f87171 (6x6 dot)

## 4. 전장 탭

두 가지 상태: 전장 선택 / 전투 중.

### 4.1 전장 선택 (비전투)

**1단계: 전장 목록**
```
┌─────────────────────────────┐

├─────────────────────────────┤
│  ┌───────────────────────┐  │
│  │ 수련장               │  │
│  │ 답파불가 · 지정사냥   │  │
│  └───────────────────────┘  │
│  ┌───────────────────────┐  │
│  │ 야산             →   │  │
│  │ ???                   │  │  (정보 숨김)
│  └───────────────────────┘  │
│  ┌───────────────────────┐  │
```

```
│  다람쥐  HP25 공4    [사냥]  │  (5회+ 처치)

│  ???                        │  (미조우, 사냥 불가)
```

### 4.2 전투 화면

답파/지정사냥 시작 시 전장 탭이 전투 화면으로 전환.

```
┌─────────────────────────────┐
│ 야산 - 답파 3/5     [포기]  │  진행도 + 포기
├─────────────────────────────┤
│                             │
│  [캐릭터]    ⚔    [적]     │  전투 장면
│                             │  배경 이미지 (야산)
│  내HP ████░   적HP ███░    │  HP 오버레이
│  134/164       58/130      │
│                             │
├─────────────────────────────┤
│  위력18 · 간격2.8초 · HP134 │  내 상태 한줄
├─────────────────────────────┤
│                             │
│  삼재검법으로 늑대에게 18 피해│  전투 로그
│  늑대가 할퀴었다! 16 피해.  │  (스크롤)
│  평타로 늑대에게 5 피해.    │
│  치명타! 늑대에게 23 피해!  │  (금색)
│  — 늑대 처치! 심득 +15 —   │  (회색)
│  — 곰 등장 —               │  (회색)
│                             │
├─────────────────────────────┤
│ [삼재검법★★][무당보법★]     │  장착 무공 칩
│ 심법: 삼재심법 ★★           │
├─────────────────────────────┤
│ [내공/경맥][무공/능력][전장][업적]│
└─────────────────────────────┘
```

#### 전투 장면 영역 (30~35%)
- 배경: 전장별 배경 이미지. 없으면 CSS gradient.
- 좌: 캐릭터 이미지 100x100. 경지별. 없으면 이모지.
- 우: 적 이미지 100x100. 없으면 이모지.
- 가운데: ⚔ (opacity 0.3).
- 하단 오버레이: 내 HP바(좌) + 적 HP바(우). 4px.
- 적 정보: 처치 횟수에 따라 이름/HP 표시. 미조우면 ???.
- 보스전: 상단에 타이머 바 추가.
- drop-shadow로 배경에서 분리.

#### 내 상태 바
- 한 줄: 위력(장착 무공 중 최대 power) · 공격 간격 · HP
- 14px weight 500.

#### 전투 로그 (핵심)
- flex:1, 스크롤.
- 공격 1회 = 로그 1줄.
- 공격 간격이 로그 속도 결정. 초반 4초에 1줄, 후반 1~2초에 1줄.
- 자동 스크롤: 새 로그 추가 시 하단으로.

**3색만 사용:**

| 색상 | 용도 |
|------|------|
| 흰색 rgba(255,255,255,0.85) | 일반 공격/피격 전부 |
| 금색 rgba(255,215,0,0.9) | 치명타, 2연타, 회피, 드롭, 보스 등장 |
| 회색 rgba(255,255,255,0.3) | 처치, 등장, 심득 |

**로그 스타일:** 12px, line-height 1.8. 금색은 weight 500. 회색은 11px.

**로그 내용 — 다양한 표현 랜덤:**

| 몬스터 | 공격 표현 (2~3종 랜덤) |
|--------|----------------------|
| 다람쥐 | '다람쥐가 재빠르게 물었다!' / '다람쥐가 도토리를 던졌다!' |
| 토끼 | '토끼가 뒷발로 찼다!' / '토끼가 돌진했다!' |
| 여우 | '여우가 꼬리를 휘둘렀다!' / '여우가 날카롭게 물었다!' |
| 사슴 | '사슴이 뿔로 받았다!' / '사슴이 돌진해왔다!' |
| 멧돼지 | '멧돼지가 이빨로 들이받았다!' / '멧돼지의 돌진!' |
| 늑대 | '늑대가 발톱으로 할퀴었다!' / '늑대가 물어뜯었다!' |
| 곰 | '곰이 거대한 앞발로 내리쳤다!' / '곰의 포효와 함께 강타!' |
| 산군 | '산군의 발톱이 번개처럼 스쳤다!' / '산군이 포효하며 덮쳤다!' |

몬스터 데이터에 attackMessages 필드로 관리.
무공별 공격 표현도 2~3종 준비 (attackMessages).

**무당보법(패시브) 회피 시 로그:**
'늑대의 공격을 가볍게 피했다!' (금색)

**평타 로그:**
'평타로 늑대에게 5 피해를 입혔다.' (흰색)

#### 장착 무공 칩 바
- 로그 아래, 탭바 위.
- 각 무공: chip (이름 + 성급별). 11px. 반투명 배경.
- 심법: 파란 계열 텍스트.

## 8. 전투 중 다른 탭 이동

- 전투 중 다른 탭으로 이동 가능. 전투는 백그라운드에서 계속 진행.
- 다른 탭에 있을 때 전투 종료 시, 전장 탭으로 돌아오면 결과 화면 표시.
- 전투 중 내공/경맥 탭: 경맥 투자 버튼 비활성.
- 전투 중 무공/능력 탭: 장착/해제 버튼 비활성.

## 9. 구현 주의사항

- 게임 로직(gameStore, 전투 계산)은 건드리지 않는다. CSS/컴포넌트만 변경.
- 전투 로그 메시지 템플릿(4.2장)은 gameStore의 로그 push에서 변경.
- 몬스터별 공격 표현(4.2장 테이블)을 몬스터 데이터에 attackMessages 추가.
- 무공별 공격 표현도 무공 데이터에 attackMessages 추가.
- 플로팅 텍스트 시스템 제거. 로그가 대체.
- Noto Serif KR 제거, Noto Sans KR 통일.
- 캐릭터 이미지(player/tier0.png) 연결 확인.
- DPS 관련 UI 표시 완전 제거. 대신 '위력'과 '공격 간격' 표시.
- frontend-design 스킬의 디자인 원칙을 참조하여 AI slop 피하기.
- GAME_GUIDE.md의 UI 원칙 준수.
