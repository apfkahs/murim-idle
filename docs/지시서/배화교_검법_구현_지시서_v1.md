### A-1. 소각 트리거 구조 수정

**핵심 원칙**: 재의 묵념/맹세는 심법 소각(자동/절초)에서만 발동. **소각된 스택 수** 기준으로 계산 (남은 스택 수 아님).

#### `app/src/utils/combat/baehwagyoEmberTick.ts`

1. `applyAshMukneom`, `applyAshMaengse`를 **export** 로 변경
2. `tickBaehwagyoEmber` 내 호출 수정:

```ts
// 기존 (잘못됨 — 남은 스택 수 사용)
applyAshMukneom(ctx, stacksBefore);
applyAshMaengse(ctx, stacksBefore);

// 수정 (소각된 수 = 1)
applyAshMukneom(ctx, 1);
applyAshMaengse(ctx, 1);
```

#### `app/src/utils/combat/playerCombat.ts`

`tryEmberUltDelete` 함수: `consumeEmberStacks` 직후 핸들러 추가.

```ts
const res = consumeEmberStacks(dots, toConsume);
if (res.consumed > 0) {
  applyAshMukneom(ctx, res.consumed);   // 추가
  applyAshMaengse(ctx, res.consumed);   // 추가
}
```

#### `app/src/utils/combat/sarajinunBulggotTick.ts`

수정 없음. 장비 소각은 핸들러 호출 없음 유지 (이미 올바름).

### A-2. 재의 묵념 (mind-t1-2) 재설계

#### HP 회복 계수 (소각된 스택 1개당 최대HP%)

| 구간 | 공식 |
|---|---|
| lv1 | 0.5% |
| lv2~9 | 0.5 + (lv-1) × 0.05% |
| lv10 | 1.0% (점프) |
| lv11~19 | 1.0 + (lv-10) × 0.1% |
| lv20 | 2.5% (점프) |
| lv21~29 | 2.5 + (lv-20) × (1.0/9)% |
| lv30 | 4.0% (점프) |

`baehwagyoEffects.ts`의 묵념 계수 계산 함수를 위 공식으로 교체.

#### 특성 (카운터 누적 방식)

| 레벨 | 소각 누적 조건 | 효과 |
|---|---|---|
| lv10 | 심법 소각 **10개**마다 | 10초간 받는 피해 **10%** 감소 |
| lv20 | 심법 소각 **8개**마다 | 10초간 받는 피해 **15%** 감소 |
| lv30 | 심법 소각 **6개**마다 | 15초간 받는 피해 **20%** 감소 |

- 구현: `ctx`에 `baehwagyoMukneomBurnCounter: number` 필드 추가
- 소각 발생마다 `counter += 소각된 수`
- `counter >= 임계값`이면 피해감소 버프 발동 + `counter -= 임계값` (나머지 유지)
- 버프 필드: `baehwagyoMukneomDmgReductBuff: { pct: number, expiresAtSec: number } | null`
- 재발동 시 타이머 갱신 (중첩 없음, 더 강한 버프로 덮어쓰기)
- 특성은 해당 레벨 이상일 때만 활성화 (lv10 미만이면 카운터 불활성)

### B-2. 동적 배율 반환 함수 (`app/src/utils/combat/baehwagyoEffects.ts`)

식화심법의 `getSikhwaQiCoeff(openLv)` 패턴으로 추가:

```ts
/** 검법 메인 노드 레벨 → gradeMult */
export function getSwordGradeMult(lv: number): number {
  if (lv <= 0) return 1.0;
  if (lv === 1) return 1.5;
  if (lv <= 9) return 1.5 + (lv - 1) * 0.05;
  if (lv === 10) return 2.1;
  if (lv <= 19) return 2.1 + (lv - 10) * (0.4 / 9);
  if (lv === 20) return 2.8;
  return 2.8;
}

/** 절초 배율 노드 레벨 → 절초 배율 */
export function getSwordUltMult(lv: number): number {
  if (lv <= 0) return 3.0;
  if (lv <= 9) return 3.0 + lv * (1 / 9);
  if (lv === 10) return 4.5;
  if (lv <= 19) return 4.5 + (lv - 10) * (1 / 9);
  if (lv === 20) return 6.0;
  return 6.0;
}

/** 검기 발현 노드 레벨 → X 배율 */
export function getSwordQiManifestX(lv: number): number {
  if (lv <= 0) return 0;
  if (lv === 1) return 3.0;
  if (lv <= 9) return 3.0 + (lv - 1) * (3 / 8);
  if (lv === 10) return 7.5;
  if (lv <= 19) return 7.5 + (lv - 10) * (1 / 3);
  if (lv === 20) return 12.0;
  return 12.0;
}

/** 검기 발현 노드 레벨 → 슬라이더 최대 소모율 (0~1 범위) */
export function getSwordQiMaxDrainRate(lv: number): number {
  if (lv <= 0) return 0;
  if (lv <= 9) return 0.05;
  if (lv <= 14) return 0.075;
  if (lv <= 19) return 0.10;
  return 0.125;
}
```

### G-1. 탐식하는 불꽃 — 검법 절초 배율 연동

**대상 파일**: `app/src/data/equipment.ts` (tamsik_bulggot_weapon), 절초 데미지 계산 경로

현재 코드에 이미 TODO 주석 존재:
```ts
// TODO: 배화교 검법 추가 시 5만 스택에서 절초배율 +1, 10만 스택에서 +1
```

**구현 내용**:
- 탐식하는 불꽃 장착 + 킬카운트 5만 이상 → 절초 배율 +1.0
- 탐식하는 불꽃 장착 + 킬카운트 10만 이상 → 절초 배율 추가 +1.0 (누적 +2.0)
- 이 보너스는 `getSwordUltMult(nodeLv)`의 반환값에 **합산**되어야 함

**절초 배율 최종 계산식**:
```
최종 절초 배율 = getSwordUltMult(ultNodeLv)
              + tamsikUltBonus(killCount)   // 5만↑: +1.0 / 10만↑: +2.0
```

- `tamsikUltBonus`는 탐식하는 불꽃 장착 여부 + 킬카운트 기준으로 결정
- 탐식하는 불꽃 미장착 시 0

### G-2. 삼행의 철칙 — 검법 장착 시 피해감소 완전 해제

**배경**: `삼행의 철칙(三行鐵則)`은 경보사와 외문수좌가 battle_start에 발동하는 패시브 패턴.
배화교 무공 미장착 시 적이 받는 피해가 **25%**만 통과 (75% 감소).
기존 배화교 무공 1종 이상 장착 시 **75%** 통과 (25% 감소).

**이번 패치 추가**: 배화교 검법 장착 시 피해감소 **완전 해제** (100% 통과).

#### 경보사 (`monsters.ts` — `gyeongbosa_iron_rule`)

현재 조건 판정 로직 (`baehwa_guard` 타입 핸들러 또는 `applyBattleStartSkills`):

| 조건 | 현재 배율 | 변경 후 배율 |
|---|---|---|
| 배화교 무공 0종 | 0.25 | 0.25 (유지) |
| 배화교 무공 1+종 (검법 제외) | 0.75 | 0.75 (유지) |
| **배화교 검법 장착** | — | **1.0 (신규)** |

조건 판정 순서: 검법 장착 여부를 먼저 확인 후 → 기존 무공 수 분기.

#### 외문수좌 (`skillHandlers/baehwagyo/oemun_suja.ts` — `oemun_suja_guard`)

기존 코드 주석에 이미 명시:
```ts
// 0종 미장착 → 0.25, 1+종 장착 → 0.75. (검법 추가 시 3종 통합 → 1.0 분기 추가 예정)
```

| 조건 | 현재 배율 | 변경 후 배율 |
|---|---|---|
| 배화교 무공 0종 | 0.25 | 0.25 (유지) |
| 배화교 무공 1~2종 (검법 제외) | 0.75 | 0.75 (유지) |
| **배화교 검법 포함 3종 이상** | — | **1.0 (신규)** |

**"검법 장착" 판정 기준**: `equippedArts`에 성화검법 art ID가 포함된 경우.
