v9 주요 변경 (v8에서 되돌린 점)
  • HUD 복원 — v8에서 해체했던 상단 HUD(포트레이트 + 이름 + 가로 HP/MP 바 + 상태칩 + DPS)를 v7 스타일로 복원. 단 포트레이트는 v7의 60x60에서 64x64로 조금 더 크게. "사진이 작다"는 문제 해소.
  • 세로 HP 레일 추가 — v8의 "세로 HP 기둥" 아이디어는 그대로 살리되, 이름·숫자·포트레이트·칩을 제거하고 순수 바만 로그 양옆에 20px 얇은 기둥으로 남김. 왼쪽 = 내 HP, 오른쪽 = 적 HP. 바닥 앵커, 위에서 아래로 깎임 — 이 메커니즘은 유지.
  • 로그 영역 회복 — v8에서 64px씩 잡아먹던 컬럼이 20px씩으로 축소. 로그 가용 폭이 가로 약 88px 회복. v7 수준의 넉넉한 로그 공간 복귀.
  • 이중 HP 표현 — HUD 가로 바(숫자 동반, 정밀한 % 읽기용)와 세로 레일(주변시로 자연스레 인지). 두 표기가 겹치지만 HP는 가장 중요한 정보이고, 서로 다른 읽기 모드에 쓰임: 가로 바는 "정확히 얼마" · 세로 레일은 "대략 얼마"의 주변 감지.
  • 25/50/75 틱 마크 — 레일 안에 가로 실선 3개로 눈금 표시. 체력 %를 숫자 읽지 않고도 즉시 파악.
  • "나"/"적" 미세 라벨 — 레일 맨 위에 작은 라벨로 좌우 소속 명시(8.5px, 글자 거의 안 띔). 불필요하면 제거 가능.
v7 → v9 요약
  • v7의 핵심(중앙 축 연장, LATEST 스트립, HP 델타, 칩 통일) 전부 유지
  • 추가된 것: 로그 양옆 세로 HP 레일 2개
  • 변경된 것: 포트레이트 60→64
조정 포인트
  • 레일 폭 — 지금 20px. 더 얇게(14-16px, 거의 안 보이게) / 더 두껍게(28-32px, 주인공급)
  • "나"/"적" 라벨 — 유지 / 제거 / 아이콘(●/●)으로 대체
  • 틱 마크 개수 — 지금 25/50/75 3단. 50%만(1단) / 10%씩(9단)도 가능
  • HUD의 가로 HP 바 — 레일이 있으니 제거 고려 가능. 단 숫자 표기는 유지해야 함 (%만으론 수치가 안 보임)
  • 포트레이트 크기 — 64px. 더 크게(72-80px)로 훨씬 키울 수도 있음(HUD 높이 증가)
  • 레일 피해 애니메이션 — 구현 시 방금 깎인 구간에 붉은 웨지 오버레이 0.4-0.8s 페이드
