v10 주요 변경 (v9 → v10)
  • HUD 가로 HP/MP 바 전부 제거 — 더 이상 HUD에 수평 바 없음. HP 시각화는 로그 양옆의 세로 레일이 전담.
  • 숫자 크게, 여유롭게 — 현재 체력 1563을 22px mono 800로 크게. 최대값 /2349와 퍼센트 67%는 작게. 계층이 명확 → "현재가 몇이냐"가 한눈에.
  • 내력(MP)은 HP 아래에 서브로 — 파란 계열 라벨, 숫자는 14px로 HP보다 작게. 플레이어 측에만.
  • 상태이상 칩 확대 — HUD 내 칩은 13px / padding 4px 12px / radius 5px로 키움. 로그 안쪽 칩(10px)과 시각적 위계 차등. 빈 상태는 — 플레이스홀더로 여백 유지.
  • 세로 레일 폭 증가 — 20px → 24px. HUD 가로 바가 없는 만큼 레일이 HP 시각화의 유일한 인디케이터가 되므로 존재감 강화. 틱 마크(25/50/75%)는 그대로.
  • 포트레이트 약간 확대 — 64px → 68px. HUD가 비워진 만큼 얼굴 공간 약간 확보.
  • 중앙 구분선 점선(dashed) 유지 — 숫자 블록과 상태칩 블록 사이에 가로 점선으로 섹션 구획.
정보 밀도 비교
  • v9 HUD: 포트레이트 + 가로 HP/MP 바(3줄) + 상태칩 + DPS — 밀도 높지만 바+숫자가 중복
  • v10 HUD: 포트레이트 + 숫자(크게) + 상태칩(크게) + DPS — 2개 차원(숫자 정밀 / 레일 주변시) 명확 분리
조정 포인트
  • 현재 체력 숫자 크기 — 지금 22px. 더 크게(28px) / 더 작게(18px)
  • 퍼센트 (67%) 표시 — 유지 / 제거 (레일 틱으로 이미 감지 가능)
  • MP 위치 — 지금 HP 아래. 대안: 포트레이트 옆 세로 미니 바 / 별도 줄 제거 후 숫자만
  • 상태 — 플레이스홀더 — 상태 없을 때 칸 비워둘지 대시로 채울지
  • 레일 폭 — 24px. 더 두껍게(30-36px)로 존재감↑ / 혹은 16px로 미니멀
  • DPS 위치 — 지금 HUD 맨 아래. 대안: 상태칩과 같은 줄에 / 로그 상단 헤더에 / 아예 제거
  • 포트레이트 크기 — 68px. 더 크게(80px)도 여유 있음
