# 배화교 스킬트리 UI 구현 (워크트리 실험)

## 1. 작업 개요

무림 방치록의 **배화교 던전 스킬트리 UI 프로토타입**을 구현한다. 이 작업은 실제 화면을 띄워보며 UI 방향을 검증하기 위한 실험적 구현이며, 메인 브랜치에 영향을 주지 않도록 **새 워크트리에서 작업**한다.

### 1.1. 목표

아래 명세에 따라 스킬트리 화면을 구현하여, 실제 렌더링된 화면을 보며 레이아웃·상호작용·시각적 톤을 검증할 수 있는 상태를 만든다.

### 1.2. 구현 범위

**포함**

- 스킬트리 화면 전체 레이아웃
- 4가지 + ??? 탭 전환
- 노드 시각화 (디아블로2 스타일, 아이콘은 플레이스홀더)
- 노드 상태 표시 (해금 / 흐림 / 블러 / 만렙)
- 노드 클릭 시 모달 팝업 (상세 / 레벨업)
- 잠긴 영역 클릭 시 해금 조건 팝업
- 환전소 UI (??? 탭 상단)
- 상단 인벤토리 표시 (자원 3종 + 비급 3종)
- 검법 가지의 1단계 세부 구조 (초식 배율 먼저, 나머지 해금)

**제외 (이번 작업에서는 구현하지 않음)**

- 실제 게임 로직 연결 (몬스터 드랍, 전투 적용 등)
- 무공 창 연동 (별도 작업)
- 애니메이션 · 이펙트 (최소한의 호버 · 페이드만)
- 수치 밸런싱 (목업용 임시 수치 사용)
- DALL-E 아이콘 (플레이스홀더 아이콘 사용)
- 심법 / 보법 / 외법의 세부 구조 (검법만 정확히 구현, 나머지는 동일 템플릿)

## 2. 시스템 구조 (배경 지식)

### 2.1. 가지 구조

- 총 5개 탭: 검법 / 심법 / 보법 / 외법 / ???
- 각 가지(??? 제외)는 8개 노드: 1단계 4개 + 2단계 2개 + 3단계 2개
- ??? 탭은 환전소 + 최종 절초 노드로 구성

### 2.2. 노드 레벨 구조

- **1단계 노드**: 만렙이 진행도에 따라 확장
  - 하위 무공 획득 시: 1~30레벨 (희미한 잔불로 찍음)
  - 상위 무공 해금 후: 31~45레벨 (타오르는 불꽃 파편으로만 찍음)
  - 최상위 무공 해금 후: 46~50레벨 (신성한 불의 정수로만 찍음)
- **2단계 노드**: 1~15 → 16~20 (확장), 최종 20레벨
- **3단계 노드**: 1~5 (확장 없음)

### 2.3. 자원

- 희미한 잔불 (일반 정수)
- 타오르는 불꽃 파편 (상위 정수)
- 신성한 불의 정수 (최상위 정수)

### 2.4. 비급

- 12종: (검법 / 심법 / 보법 / 외법) × (1단계 / 2단계 / 3단계)
- 가지 내에서만 이동 가능 (검법 1단계 비급은 검법 1단계 노드 어디든 사용 가능)
- 노드 1포인트 사용 = 정수 N개 또는 비급 1권

### 2.5. 정가(천장) 경로 해금

- **2단계 해금**: 1단계 노드 2개 이상 20레벨 + 희미한 잔불 대량
- **3단계 해금**: 2단계 노드 1개가 12레벨 이상 + 타오르는 불꽃 파편 대량
- 또는 상위 무공 비급서 획득 시 즉시 해금

## 3. UI 명세

### 3.1. 전체 레이아웃

```
┌─────────────────────────────────────────┐
│  [검법] [심법] [보법] [외법] [???]       │  ← 탭 바
├─────────────────────────────────────────┤
│  🔥 희미한 잔불 1,234 │ 📜 검법1단계 3   │  ← 인벤토리
│  🔥 타오르는 불꽃 45   │ 📜 검법2단계 1   │
│  🔥 신성한 불의 정수 2  │ 📜 검법3단계 0  │
├─────────────────────────────────────────┤
│                                          │
│          (트리 영역, 세로 스크롤)         │
│                                          │
└─────────────────────────────────────────┘
```

- 탭 바와 인벤토리는 상단 고정 (sticky)
- 트리 영역만 세로 스크롤
- ??? 탭 선택 시 인벤토리의 비급 영역은 숨김 (자원 3종만 표시)

### 3.2. 검법 가지 트리 구조

```
        [초식 배율]        ← 0단계 해금 노드 (유일한 최초 활성 노드)
            │
    ┌───────┼───────┐
  [절초]  [치확]  [치피]    ← 초식 배율 1레벨 찍으면 동시 해금
  
  ─────────────────────    ← 1단계와 2단계 사이 구분선
  
  (블러 처리 영역)
  [절초 강화 1] [공격속도]   ← 2단계
  
  ─────────────────────
  
  (블러 처리 영역)
  [절초 강화 3] [절초 강화 4] ← 3단계
```

### 3.3. 노드 상태

각 노드는 다음 상태 중 하나:

1. **찍을 수 있음**: 정상 표시, 클릭 시 상세 팝업
2. **찍을 수 없음 (자원 부족)**: 정상 표시지만 팝업의 레벨업 버튼 비활성
3. **흐림 (선행 노드 미충족)**: 흐리게 표시, 클릭 시 팝업은 뜨되 레벨업 불가, 안내 문구
4. **만렙 도달**: 아이콘 주변 빛 이펙트 (glow 효과)
5. **블러 영역 (단계 잠김)**: 구역 전체 블러 + 오버레이 문구, 클릭 시 해금 팝업

### 3.4. 노드 박스 디자인

- 디아블로2 스타일 참고 (돌 · 양피지 질감, 사각형 아이콘 박스)
- 각 노드 옆에 작은 숫자 박스 `현재/만렙` 표시 (예: `15/30`)
- 아이콘은 **플레이스홀더** (색상 박스 + 노드 이름 약자 정도)
- 단계별 색상 톤:
  - 1단계: 붉은색 계열
  - 2단계: 진한 적색 + 금선
  - 3단계: 검붉은색 + 금광

### 3.5. 모달 팝업 (노드 상세)

```
┌─────────────────────────────────┐
│  [X]                            │
│                                 │
│  [아이콘]  염화검법 - 초식 배율  │
│           현재 25 / 30           │
│                                 │
│  ───────────────────────────    │
│                                 │
│  염화검법 초식의 위력을 높인다.  │  ← 서사 한 줄
│  (공격 시 배율 증가)             │  ← 기능 한 줄
│                                 │
│  ───────────────────────────    │
│                                 │
│  현재: +45%                     │
│  다음: +48%  (▲ +3%)            │
│                                 │
│  ───────────────────────────    │
│                                 │
│  자원 선택:                      │
│  [ 🔥 희미한 잔불 50개 사용 ]    │
│  [ 📜 검법 1단계 비급 1권 사용 ] │
│                                 │
│         [ 레벨업 ]              │
└─────────────────────────────────┘
```

- 배경 어두워짐 (dim overlay)
- X 버튼 또는 바깥 클릭으로 닫힘
- 자원 선택 버튼 2개: 정수 / 비급
- 보유량 부족한 옵션은 비활성화
- 만렙 도달 시 "다음" 행 대신 "만렙 도달" 표시
- 흐림 노드는 효과 / 자원 영역 대신 "먼저 초식 배율을 1레벨 이상 투자하라" 안내


## 4. 기술 요구사항

### 4.1. 플랫폼

- **React + TypeScript** (무림 방치록 기존 스택과 동일)
- 기존 프로젝트 구조 · 컨벤션 준수
- 기존 무공 창과 **별도 컴포넌트 계층**으로 구성 (스킬트리는 독립 모달 / 화면)

### 4.2. 상태 관리

- 목업용 임시 상태 (useState로 충분)
- 다음 값들을 상태로 관리:
  - 선택된 탭
  - 각 노드의 현재 레벨
  - 자원 3종 보유량
  - 비급 12종 보유량
  - 상위 / 최상위 무공 해금 여부 (가지별)
  - ??? 편 획득 여부 (첫 번째 / 두 번째)
  - 선택된 노드 (모달용)

### 4.3. 반응형

- PC와 모바일 모두 지원
- 모바일에서 탭 바는 가로 스크롤 또는 축약 고려

### 4.4. 스타일

- 기존 무림 방치록의 톤앤매너 참고
- 배화교 전용 톤: 검 · 적 · 금 3색 대비 (양피지 / 돌 질감)
- 기존 무공 UI와 시각적으로 구분되게

## 8. 주의 사항

- **수치는 전부 임시값**. 밸런싱은 별도 작업이므로 수치 자체에 대한 판단 금지.
- **아이콘은 플레이스홀더**. DALL-E 생성은 나중에.
- **기존 코드를 건드리지 말 것**. 이 작업은 완전히 독립된 새 화면으로 구성.
- **불확실한 디자인 판단은 추측하지 말고 명세대로** 구현. 명세에 없는 장식적 요소 추가 금지.
- 작업 중 명세에서 애매하거나 모순된 부분 발견 시 **바로 멈추고 질문**.
