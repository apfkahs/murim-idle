## 1. 전체 컨셉

배화교 던전은 기존과 다른 **반복 플레이로 강해지는 체감**을 주는 던전.

- **기존 무공**: 사냥 → 숙련도 자동 누적 → 수동적 성장
- **배화교 무공**: 사냥 → 정수 드랍 → 정수를 트리 노드에 직접 투입 → 능동적 성장

"배화교 무공 = 사교의 비전, 공력을 직접 새겨야 하는 이단의 무공"이라는 서사로 기존 무공과 차별화.

## 2. 자원 시스템

### 2-1. 정수 3종 (환전 가능)

| 자원 | 이름 확정 | 주요 획득처 | 사용 구간 |
|------|---------|-----------|---------|
| 일반 정수 | **희미한 잔불** | 외문~내문 몬스터 | 1단계 노드 1~30레벨 |
| 상위 정수 | **타오르는 불꽃 파편** | 사원 이상 몬스터 | 1단계 31~45레벨, 2단계 1~15레벨 |
| 최상위 정수 | **신성한 불의 정수** | 심처 몬스터 | 1단계 46~50레벨, 2단계 16~20레벨, 3단계 1~5레벨 |

**환전 (비대칭 비율, 수치는 예시)**:
- 상승: 희미한 잔불 400개 → 타오르는 불꽃 파편 20개 → 신성한 불의 정수 1개 (20:1씩)
- 하강: 신성한 불의 정수 1개 → 타오르는 불꽃 파편 5개 → 희미한 잔불 25개 (1:5씩)
- 왕복 시 약 94% 증발 구조로 "직접 드랍을 노려라"는 압박 유지
- 남는 상위 정수를 하위로 내릴 수 있는 여지는 제공

### 2-2. 비급 (단계 × 가지 공용 토큰)

**v0.3에서 v0.4로 변경**: "노드 전용 1포인트 토큰"에서 **"가지별 단계별 공용 토큰"**으로 변경.

- 총 **12종**: (검법 / 심법 / 보법 / 외법) × (1단계 / 2단계 / 3단계)
- 예: "검법 1단계 비급"은 검법 가지의 **어느 1단계 노드에든** 1포인트 사용 가능
- 가지 간 이동 불가 (검법 비급을 심법에 쓸 수 없음)
- 단계 간 이동 불가 (1단계 비급을 2단계에 쓸 수 없음)
- 비급 1개 = 정수 N개 = 1포인트 (N값 TBD)

## 3. 드랍 시스템

### 3-1. 일반 사냥 드랍

| 드랍 | 방식 |
|------|------|
| 정수 | 저확률 드랍 (확정 아님) |
| 비급 | 저확률 드랍 (정수보다 더 낮음) |

### 3-2. 누적 보상 (업적 천장)

좌절 방지용 안전망. 배화교 몬스터 N마리 처치 시 정수 확정 지급.

### 3-3. 최초 클리어 보상

단계 최종 보스(외문 보스, 내문 보스 등) 최초 처치 시 대량 정수 지급.

- 사원 보스: 타오르는 불꽃 파편 대량
- 심처 보스: 타오르는 불꽃 파편 + 신성한 불의 정수
- 구체적 수치 TBD
- 1회성 보상 (기존 업적 창의 일반 업적 섹션 활용)

### 3-4. 구현 방침

- **정수/비급 누적 보상** → 기존 업적 시스템 반복 업적 섹션

## 4. 트리 구조 (배화교 비급)

### 4-1. 전체 구조

배화교 비급 1개 안에 **4개 가지 + 최종 절초 노드** 존재.

**v0.3에서 v0.4로 변경**: 권법 가지 제거, 5가지 → 4가지.

```
[배화교 비급]

  ┌─── 검법 가지 ────┐
  ├─── 심법 가지 ────┤
  ├─── 보법 가지 ────┤
  └─── 외법 가지 ────┘  (여러 무공 포함 예정)
  
  ═══════════════════
         [???]
  (비급 전체의 최종 절초)
  ═══════════════════
```

### 4-2. 무공 단계 (가지마다)

각 가지는 3단계 무공 진화 구조.

| 단계 | 예시 (검법 가지) | 해금 방식 |
|------|----------------|----------|
| 하위 무공 | 염화검법 | 비급 획득 후 **초식 배율 노드 1레벨 투자 시** (v0.4 변경) |
| 상위 무공 | 적염신검법 | 해금 조건 2가지 (비급 or 정가) |
| 최상위 무공 | (이름 TBD) | 해금 조건 2가지 (비급 or 정가) |

**v0.3에서 v0.4로 변경**: "비급 획득 시 기본 장착"에서 **"정수로 초식 배율 1레벨 투자 시 무공 창 등록"**으로 변경.

**하위→상위 진화 시**:
- 기존 무공의 이름이 변경되어 UI에 표시 (염화검법 → 적염신검법)
- 새로운 노드(2단계)가 개방됨
- 기존 1단계 노드의 추가 레벨도 개방됨

### 4-3. 노드 티어 구조

각 가지 내부에 노드들이 3티어로 분산.

| 티어 | 개수 (가지당) | 만렙 구조 | 해금 조건 |
|------|-------------|---------|----------|
| 1단계 노드 | 4개 | 30 → +15 → +5 = **최종 50** | 하위 무공 획득 시 |
| 2단계 노드 | 2개 | 15 → +5 = **최종 20** | 상위 무공 해금 시 |
| 3단계 노드 | 2개 | **5 (고정)** | 최상위 무공 해금 시 |

**1단계 노드의 레벨 확장 메커니즘**:

```
  하위 무공 획득:   1~30레벨    (희미한 잔불로 찍음)
  상위 무공 해금:   31~45레벨   (타오르는 불꽃 파편으로만 찍음)
  최상위 무공 해금: 46~50레벨   (신성한 불의 정수로만 찍음)
  
  최종 만렙: 50레벨
```

**2단계 노드의 레벨 확장 메커니즘**:

```
  상위 무공 해금:   1~15레벨    (타오르는 불꽃 파편으로 찍음)
  최상위 무공 해금: 16~20레벨   (신성한 불의 정수로만 찍음)
  
  최종 만렙: 20레벨
```

**3단계 노드**:

```
  최상위 무공 해금: 1~5레벨 (신성한 불의 정수로 찍음)
  
  최종 만렙: 5레벨 (확장 없음)
```

### 4-4. 상위 단계 해금 조건 (v0.4 신규 확정)

**2단계 해금 조건 (둘 중 택1)**:
- **비급 경로**: 상위 무공의 비급서 드랍 획득
- **정가(천장) 경로**: 1단계 노드 2개 이상 20레벨 달성 + 희미한 잔불 대량 소모

**3단계 해금 조건 (둘 중 택1)**:
- **비급 경로**: 최상위 무공의 비급서 드랍 획득
- **정가(천장) 경로**: 2단계 노드 1개가 12레벨 이상 달성 + 타오르는 불꽃 파편 대량 소모

정수 대량 소모 구체 수치 TBD.

### 4-5. 노드 효과 구조

**기존 무공**:
- 삼재검법 심득 1단계 = 치피 + 배율 + 절초 등이 한 덩어리로 묶여서 해금
- 통합형 패키지

**배화교 무공**:
- 무공 1개가 여러 노드로 분산
- 각 노드가 개별 효과 담당 (치피 노드 / 배율 노드 / 절초 노드 등)
- 분산형 설계로 개별 튜닝 가능

### 4-6. 검법 가지 노드 구성 (v0.4 확정)

**v0.3에서 v0.4로 변경**: 검법만 특수 구조. 1단계에 "절초", 2단계에 "공격속도" 포함.

```
1단계 (4개):
  - 초식 배율         ← 0단계 해금 노드 (유일한 최초 활성)
  - 절초              ← 찍으면 무공에 절초 기술 추가
  - 치명타 확률
  - 치명타 피해

2단계 (2개):
  - 절초 강화 1
  - 공격 속도

3단계 (2개):
  - 절초 강화 3
  - 절초 강화 4
```

**0단계 해금 메커니즘**:
- 비급 획득 후 초기 상태: **초식 배율 노드만 활성**, 나머지 3개는 흐림
- 초식 배율 1레벨 투자 시:
  - 무공 창에 **"염화검법" 등록**
  - 나머지 3개 노드(절초, 치확, 치피) **동시 해금**
- 절초 노드 1레벨 투자 시: **절초 기술이 무공에 추가됨** (등록은 초식 배율 찍을 때, 절초 기능은 절초 찍을 때)

**다른 가지(심법, 보법)는 유사한 구조 예정**. 외법은 여러 무공을 포함하므로 별도 설계.

### 4-7. 심법 / 보법 / 외법 노드 구성 (임시)

현재는 원래 구조 유지, 추후 개별 설계:

```
1단계 (4개):
  - 초식 배율
  - 치명타 확률
  - 치명타 피해
  - 공격 속도

2단계 (2개):
  - 절초 강화 1
  - 절초 강화 2

3단계 (2개):
  - 절초 강화 3
  - 절초 강화 4
```

※ 이 구성은 임시. 심법/보법도 검법처럼 "0단계 해금" 구조로 전환될 가능성 높음. 외법은 별도 설계.

## 5. 노드 레벨업 비용 구조

### 5-1. 비용 증가 원칙

- 1포인트 투입 = 정수 N개 또는 비급 1개 소모
- 레벨이 올라갈수록 **1레벨당 필요 포인트 증가**
- 후반 레벨에 갈수록 노가다 압박 극대화
- 구체적 수치 TBD (단, "답파 약 1만 회 수준의 극한 노가다"를 기준으로 설계)

### 5-2. 빌드 선택의 무게

- 노드 만렙이 매우 무거우므로 **모든 노드를 다 만렙 찍는 것은 사실상 불가능**
- 플레이어는 빌드에 맞는 일부 노드만 집중 투자
- 리스펙(재분배) 시스템 도입 여부 및 정책 TBD

## 6. 정상 경로 vs 노가다 경로

### 6-1. 정상 경로

- 상위 던전(내문/사원/심처) 진행
- 상위 비급/재료 획득으로 상위 무공 빠르게 해금
- 상위 정수 직접 드랍으로 효율적인 노드 성장
- 대부분의 플레이어가 선택하는 경로

### 6-2. 노가다 경로

- 외문에서 극한의 노가다
- 일반 정수만으로 환전을 통해 상위 노드까지 성장 가능
- **정가(천장) 경로**로 상위 단계 해금도 가능
- "상위 던전이 너무 어려워도 포기하지 않게 해주는" 우회 경로

### 6-3. 밸런스 원칙

- 정상 경로가 **시간 효율적으로 압도적**이어야 함
- 노가다 경로는 "가능성 자체가 의미"
- 시간 차이로 조절 (상위 던전 = 고효율, 외문 노가다 = 저효율이지만 결국 도달 가능)

## 7. 초기 진입 서사 (v0.4 신규)

- 배화교 던전 첫 답파 시 **적 조우 없음**
- 꿈 같은 공간 연출
- 채팅 로그 출력: **"꿈 같은 공간을 헤쳐나가다 문득 눈을 뜨니, 백지로만 가득한 비급이 손에 있었다"** (정확한 문구는 조정 가능)
- 이후 배화교 비급이 확정 드랍되어 인벤토리 편입
- 이때부터 스킬트리 화면 진입 가능

## 8. UI/UX 설계 (v0.4 신규 대섹션)

### 8-1. 스킬트리 화면 레이아웃

- 상단 탭: **[검법] [심법] [보법] [외법] [???]**
- 상단 인벤토리 (sticky): 자원 3종 + 현재 가지의 비급 3종 (가로 페어링, 세로 3줄)
  - ??? 탭에서는 비급 영역 숨김, 자원 3종만
- 세로 스크롤 트리 영역
- 디아블로2 스타일 참고 (돌/양피지 질감, 사각형 아이콘 박스)
- 단계 간 화살표 연결, 같은 단계 내 병렬 (검법 1단계는 흐림/해제로 순차 해금)

### 8-2. 노드 박스

- 아이콘은 추후 DALL-E로 생성 예정 (현재는 플레이스홀더)
- 아이콘 옆에 `현재/만렙` 숫자 박스 (예: `15/30`)
- 단계별 색상 톤:
  - 1단계: 붉은색 계열
  - 2단계: 진한 적색 + 금선
  - 3단계: 검붉은색 + 금광

### 8-3. 노드 상태 표시

1. **찍을 수 있음**: 정상 표시, 클릭 시 상세 팝업
2. **찍을 수 없음 (자원 부족)**: 정상 표시, 팝업의 레벨업 버튼만 비활성
3. **흐림 (선행 노드 미충족)**: 흐리게 표시, 클릭 시 안내 팝업
4. **만렙 도달**: **아이콘 주변 빛 이펙트** (색상 불변)
5. **블러 영역 (단계 잠김)**: 구역 전체 블러 + 오버레이 문구, 클릭 시 해금 팝업

### 8-4. 블러 영역 연출

- CSS blur + 반투명 어두운 오버레이
- 오버레이 문구: **"강적이 품은 비밀, 혹은 끊임없는 수련"**
- 해금 조건 달성 시 "안개가 걷히는 연출"로 공개

### 8-5. 모달 팝업 (노드 상세)

- 배경 어두워짐 + X 버튼 + 바깥 클릭 닫기
- 구성: 제목 + 레벨 + 서사 한 줄 + 기능 한 줄 + 현재/다음 효과 + 증가폭 + 자원 사용 선택(정수/비급) + 레벨업 버튼
- 단일 레벨업 버튼 (연속 레벨업 없음)
- 만렙 도달 시 "다음" 행 대신 "만렙 도달" 표시

### 8-6. 모달 팝업 (잠긴 영역)

- 블러 영역 클릭 시 호출
- 구성: "잠겨있다" 안내 + 해금 방법 2가지
  - 비급 경로: 상위 무공의 비급서 획득 안내
  - 정가(천장) 경로: 현재 진행도 + 필요 자원 + 실행 버튼
- 조건 미달 시 해금 버튼 비활성

### 8-7. ??? 탭 레이아웃

**해금 전 (기본 상태)**:
- 상단: **정수 환전소**
- 하단: ??? 노드 (어둠에 잠김) + 편 획득 상태 + 서사 문구

**해금 후**:
- 상단: 정수 환전소 (그대로 유지)
- 하단: 최종 절초 노드 (정식 노드로 전환)

**편 획득 상태 표시 (하이브리드)**:
- 미획득: "첫 번째 편 / 두 번째 편"으로 모호 표시
- 획득 후: "???편 (상) / ???편 (하)"로 명확 표시

**하단 서사 문구**:
```
언젠가 모든 비밀이 밝혀지면
신화가 모습을 드러내리라.
```

### 8-8. 환전소 UI

- 환전 블록 4개: 상승 2개 + 하강 2개 (그룹 구분)
- 각 블록: 자원 쌍 + 비율 + 보유량 + 배수 버튼 `[x1] [x5] [x10] [최대]`
- 보유량 부족한 배수 버튼은 비활성화
- 확인창 없음 (배수 선택이 이미 명확한 수량)

### 8-9. 진행 피드백

- **드랍 알림**: 기존 게임과 동일하게 **채팅창 로그**에만 표시
- **천장 진행도**: 기존 업적 창의 **반복 업적 섹션** 재활용
- **??? 해금 단서**: ??? 탭 하단 하이브리드 표시

### 8-10. 무공 창 연동

- 배화교 무공(염화검법 등)은 **기존 무공 창에 삼재검법, 녹림권과 동일 위상으로 편입**
- 초식 배율 1레벨 투자 시 무공 등록
- 절초 노드 1레벨 투자 시 절초 기술 추가
- 스킬트리에서 찍은 모든 노드의 합산 효과가 무공 창에 반영
- 실사용 정보 확인은 무공 창에서
